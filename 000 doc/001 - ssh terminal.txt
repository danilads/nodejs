00 - HotKeys
   -  control + C - закрыть выполнение

01 - Первый вход
   - ssh root@your_server_ip

02 - Добавление "юзера" (можно делать только после входа) имя_Пользователя@your_server_ip
   - adduser имя_Пользователя

03 - Предоставление прав администратора - "юзеру" (Это позволит "юзеру" запускать команды с правами администратора путем добавления слова sudo перед каждой командой)
   - usermod -aG sudo имя_Пользователя

04 - Смена пароля (Запускать после того, как зашли)
   - passwd

05 - Слушать порты
   - netstat -tulpn | grep LISTEN

06 - Task Manager (нагрузка) htop - красивый интерфейс
   - top
   - htop

07 - Завершить процесс — завершить процесс с pid=1234 (посмотреть в Task Manager)
   - kill 1234 

08 - Cменить текущий каталог на /etc/nginx
   - cd /etc/nginx
