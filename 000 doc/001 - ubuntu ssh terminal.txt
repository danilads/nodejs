00 - HotKeys
   - control + C - закрыть выполнение

01 - Выход
   - logout

02 - Вход
   - ssh root@your_server_ip

03 - Добавление "юзера" (можно делать только после входа) имя_Пользователя@your_server_ip
   - adduser имя_Пользователя

04 - Удаление "юзера"
   - deluser имя_Пользователя

05 - Предоставление прав администратора - "юзеру" (Это позволит "юзеру" запускать команды с правами администратора путем добавления слова sudo перед каждой командой)
   - usermod -aG sudo имя_Пользователя

06 - Смена пароля (Запускать после того, как зашли)
   - passwd

07 - Слушать порты
   - netstat -tulpn | grep LISTEN

08 - Task Manager (нагрузка) htop - красивый интерфейс
   - top
   - htop

09 - Завершить процесс — завершить процесс с pid=1234 (посмотреть в Task Manager)
   - kill 1234 

10 - Cменить текущий каталог на /etc/nginx
   - cd /etc/nginx 

11 - Установка от имени "Супер Юзера" sudo (npm/node/git)
   - sudo apt install nodejs
